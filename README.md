## Project Overview

Developed a TODO Management Backend Application using **FastAPI** integrated with **PostgreSQL Database** to Manage task-related operations through REST APIs.

The application provides CRUD operations for TODO tasks with persistent database storage.

---

## Work Done
- Designed backend architecture using FastAPI framework.
- Implemented REST API endpoints for TODO management.
- Integrated PostgreSQL database for persistent database storage.
- Configured database connection using SQLAlchemy ORM.
- Created database models for TODO table Structure.
- Implemented Create, Read, Update and Delete (CRUD) operations.
- Established database session handling using dependency injection.
- Tested API endpoints using FastAPI Swagger UI.
- Verified request and response handling through API documentation.
- Stored and retrieved TODO data from PostgreSQL database successfully.

---

## API Functionalities Implemented

- Create TODO tasks
- Retrieve all TODO items
- Update existing TODO tasks
- Delete TODO tasks
